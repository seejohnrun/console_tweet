Console Tweet

tweet from your console, because fancy interfaces are overrated

Getting started

To get started install the gem:

$ gem install console-tweet

And authenticate with Twitter:

$ twitter setup

Type the pin shown on Twitter and that's it. Now you can use any available command.

Commands

Send a tweet:

$ twitter tweet 'Message'

View your timeline since last view:

$ twitter

View most recent replies and mentions since last view:

$ twitter replies

View most recent status:

$ twitter status
